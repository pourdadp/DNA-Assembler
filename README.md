# 🧬 DNA Overlap Assembler

A browser-based tool for assembling DNA sequences using overlap-layout-consensus (OLC) method with greedy algorithm.

## 🔬 Features
- Greedy overlap assembly with longest-to-shortest read sorting
- Reverse complement checking for overlaps
- Graphical visualization of overlaps (green = overlap, yellow = mismatch)
- Configurable minimum overlap length and maximum consecutive mismatches
- Sample test data included
- Mobile-friendly interface

## 🚀 Live Demo
[Click here to use the assembler](https://pourdadp.github.io/DNA-Assembler/index html)

## 🛠️ Tech Stack
- HTML5, CSS3, Bootstrap 5, JavaScript
- No server required – runs entirely in the browser
